# Password Manager Project Description

The Password Manager is a Java-based desktop application developed using Swing GUI components. The main objective of this project is to generate secure and strong passwords based on user-specific details such as name, PAN number, and date of birth. The application improves password security by using the SHA-256 hashing algorithm along with Base64 encoding techniques.

The system provides a user-friendly graphical interface where users can enter their personal details and automatically generate a unique password. The generated password includes uppercase letters, numbers, and special characters to enhance security strength. The application also evaluates the generated password and classifies it as Strong, Medium, or Weak based on predefined conditions.

This project demonstrates the practical implementation of Java concepts such as GUI programming, event handling, string manipulation, hashing algorithms, exception handling, and object-oriented programming. It can be used as a basic real-life utility application for secure password generation and management.

## Features

* User-friendly Java Swing interface
* Automatic secure password generation
* SHA-256 encryption for better security
* Password strength checker
* Use of special characters, digits, and uppercase letters
* Error handling for incomplete input fields

## Technologies Used

* Java
* Swing (GUI)
* AWT Layouts
* SHA-256 Hashing
* Base64 Encoding

## Conclusion

The Password Manager project helps users create secure passwords efficiently while demonstrating important Java programming concepts. It is a practical and innovative application that combines security techniques with GUI-based software development.
